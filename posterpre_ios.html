Expert-AI Collaboration: Empowering Enterprise Project Decision-making

I'm Jiarui Zhang, currently studying Computer Science at the University of Sydney. This project aims to develop an expert-AI collaborative intelligent decision-making platform, leveraging cutting-edge technologies to digitally transform the decision-making process in enterprise project management.

Project Brief

  • This project has developed an expert-AI collaborative intelligent decision-making platform for enterprise project management.
  • Integrated cutting-edge AI reasoning and RAG (Retrieval-Augmented Generation) technology.
  • The system supports multi-modal intelligent Q&A and dynamic quality assessment.
  • Capable of efficiently parsing and understanding complex project documents, providing precise and explainable decision recommendations in real-time.

Technology Stack Overview

The system is based on Flask Web framework, integrated with OpenAI GPT-4 model, utilizing LightRAG framework for multi-modal retrieval, combined with vector database and knowledge graph.

The system supports two query modes: Hybrid mode (Mix mode) (fast response) and Best mode (Best mode) (multi-modal scoring).

Why Expert-AI Collaboration?

Addressing Complex Project Decision Challenges

Project Decision Challenges

In large enterprise projects, decision-making processes are typically complex and time-critical, requiring the processing of massive documents and extracting accurate, reliable information. Traditional project management approaches face efficiency bottlenecks in information retrieval and decision support.

AI-Powered Opportunities

AI technology, particularly when combined with RAG and large language models, can efficiently parse and understand complex project documents. By deeply integrating AI's powerful processing capabilities with expert experiential knowledge, we can significantly enhance the intelligence level and response efficiency of decision-making.

This project aims to help enterprises achieve comprehensive digital transformation through AI empowerment, gaining data-driven insight support in the decision-making process.

Future Prospects

Project Potential Directions

This project will undergo continuous iteration, with future research focusing on the following three core aspects, aiming to further enhance the AI chatbot's performance, intelligence, and user experience:

  • Reinforcement Learning-driven Answer Optimization
    This direction aims to enable the AI chatbot to autonomously learn and optimize its answer generation strategies by introducing Reinforcement Learning mechanisms. We will design multi-dimensional rewards (including factual accuracy, information completeness, conciseness, fluency, relevance, citation quality, and user feedback). A Reinforcement Learning Agent will be utilized for information selection, answer reconstruction, memory integration, and citation strategy learning, thereby comprehensively improving answer quality.
  • Machine Learning-driven Precise Citation
    This direction focuses on enhancing the intelligence and precision of answer citations, ensuring that every piece of information has a clear, verifiable source. We will leverage deep embeddings for semantic matching to achieve context-aware and intelligent citations. Research areas include intent recognition, cross-document linking, granularity prediction, and citation style generation, ensuring the professionalism and traceability of references.
  • Intelligent Memory Integration
    This direction aims to elevate the chatbot's "memory" capabilities, enabling it to more effectively utilize historical information to enhance answer quality and conversational coherence. We will construct memory as a controllable knowledge base (including long-term conversational context, user preferences, and project-specific rules). Through advanced encoding, dynamic retrieval, and gating mechanisms, we will achieve intelligent memory fusion, thereby generating more personalized and coherent answers.
